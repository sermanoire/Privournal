# Privournal

I'm making a cool Journal Encryptor! For me, someone reading my journal is a legit fear. So with Privournal (Private + Journal, Hehe), I can basically write my journal on any app (I use notes on Mac), copy the whole thing, and feed it into Privournal to encrypt it. Then, I can copy the encrypted text and paste it back into the app where I keep my journals.

I’ll keep a lot of encryption options. It'll start with the basic ones, where each letter will be assigned a number  (I'll call it a "cover"). Or, one can select custom encryption and manually set each letter’s cover to a number, a symbol, another letter (Or a combination of all!).

Inside custom encryption, I'm also gonna add what I call the "Swiption" feature. With Swiption, it’s super hard to crack the encryption. You can set the letter A's cover to 2 or more different covers! (Like #,$,%) And you can then choose the exact occurrence at which they switch covers, like dynamically changing the cover every 4th time 'A' appears in the Journal! 

My future plans include making this into a website and then even like a tiny app that appears in the menu bar on top.
